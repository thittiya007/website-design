การจัดการเลย์เอาท์เว็บเพจด้วย Position

    การจัดตำแหน่งด้วย position สามารถใส่ค่าต่างๆ ได้ดังนี้
  • static ค่าปกติ default
  • relative จัดวางตามตำแหน่งตามที่กำหนด
  • absolute จัดวางตามเอลิเมนต์แม่
  • fixed อยู่ในตำแหน่งเดิม
  • sticky ติดขอบเมื่อเลื่อนไปถึง
